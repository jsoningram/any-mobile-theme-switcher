== Description ==

This plugin was devloped for one of the project of mine. Found many plugins for theme switcher but they were not able to detect all the mobile browsers. And thought to add here to help others.

Detects all the Mobile Browsers.

* Iphone / Ipod
* Ipad
* Parm Os
* Android
* Windows Mobile
* Opera Mini
* Others Mobile Browsers

You can select a different mobile theme for each mobile browser. You can select different themes for each of the options above.

This plugin is still a child. Please report any bugs @ http://dineshkarki.com.np/any-mobile-theme-switcher/report-bugs.

== Installation ==

1. Upload the theme files to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Select Themes For Mobile Browsers under Settings Menu
